# Scamazon (Parody site)

How to run:

1. Create a virtual environment and install dependencies:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. Run the app:

```bash
python app.py
```

Then open http://127.0.0.1:5000 in your browser.
